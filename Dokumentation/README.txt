STARTEN DES PROJEKTES
------------
 
 * Das Projekt steht in einem Docker Container zur verfügung.
 * Das Projekt muss nur mit dem Befehle 'docker-compose up --build' gestartet werden.
 * Eine Internetanbindung ist notwendig. Postleitzahl Daten werden von einer Online API abgerufen.
 * Initial wurde bereits ein Admininistrator und ein Nutzer Account angelegt. Weitere Nutzer können bei Laufzeit angelegt werden.
 * Daten zu Autovermietung wurden ebenfalls angelegt. Darunter ausgeliehene und zurückgegebene Autos, Hersteller und Modele.
